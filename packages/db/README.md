# @project-cvsa/db

## Available Scripts

- `gen` – Generate Prisma client and schema artifacts
- `deploy` – Deploy database changes
- `migrate` – Create a new migration file without applying it
- `migrate:dev` – Apply migrations to the development database
- `push` – Push schema changes directly to the database
- `test:setup` – Reset and apply migrations for test environment
- `studio` – Open Prisma Studio database GUI
